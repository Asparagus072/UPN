# EventLink

**EventLink** je spletna platforma, kjer lahko uporabniki ustvarjajo, objavljajo in promovirajo svoje dogodke â€“ od zabav in koncertov do poslovnih sestankov in druzenj. Udelezenci se lahko na dogodke prijavijo.

---

## Funkcionalnosti

- Ustvarjanje dogodkov
- Prijava na dogodke glede na interese, lokacijo, starost ipd.
- Filtriranje in iskanje dogodkov

---

## Tehnologije

- **Backend:** Python + Flask
- **Frontend:** HTML, CSS (Bootstrap), Jinja2 templating, JavaScript
- **Baza podatkov:** SQLite 
- **Razvojno okolje:** Visual Studio Code
